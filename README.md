# 导航页

一个简洁的上网导航和搜索入口，支持后台管理链接、分组、搜索引擎、收录申请和基础站点配置。

## 安装

1. 将源码上传到网站根目录。
2. 访问 `http://域名/install`，按提示配置数据库并完成安装。
3. 后台地址为 `http://域名/admin`，默认账号密码为 `admin` / `123456`。
4. 登录后请立即修改后台账号密码。

## 常用入口

- 前台：`http://域名/`
- 后台：`http://域名/admin`
- 收录申请：`http://域名/apply`

## 功能

- 后台管理导航链接、分组和搜索引擎。
- 支持自定义站点标题、Logo、背景和备案号。
- 支持用户提交收录申请。
- 支持详情页和直接跳转两种访问方式。

## 背景

可在后台配置背景图片。若需要使用 Bing 每日图片，可配置 `assets/img/cron.php` 并添加定时任务：

```text
GET http://域名/assets/img/cron.php
```

任务执行后会将图片保存到 `assets/img/background.jpg`，再在后台把背景地址设置为 `./assets/img/background.jpg`。
